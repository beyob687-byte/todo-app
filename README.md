📝 To-Do App

A simple and efficient To-Do application built with HTML, CSS, and JavaScript to help users manage daily tasks and stay organized.

🚀 Features
➕ Add new tasks
✅ Mark tasks as completed
❌ Delete tasks
💾 Data persistence using Local Storage
📱 Responsive design for all devices
🧠 How It Works
Users can add tasks through an input field
Tasks are dynamically rendered on the page
Each task can be marked as completed or removed
Tasks are saved in localStorage, so they remain after refreshing
